Объекты Windows

  1. Контекстное меню, это: ...

    Меню, появляющееся после нажатия кнопки Пуск.
    Меню, появляющееся после щелчка правой кнопкой мышки по выбранному объекту
    Меню, расположенное в верхней части окна приложения.
    Меню, появляющееся после двойного щелчка по выбранному объекту.

  2. Текстовый редактор Блокнот (Notepad) позволяет:

    Просматривать текстовые файлы.
    Изменять текстовые файлы
    Создавать текстовые файлы.
    Все вышеперечисленное

  3. Ярлык - это:

    Представитель папки, устройства, файла с документом или программы.
    Цветной рисунок с надписью.
    Программа, запускающая программы обслуживания.
    Правильного ответа нет.

  4. Кнопки на панели задач показывают: ...

    Какие программы в данный момент открыты.
    Какие файлы заражены вирусом.
    Какие программы зависли.

  5. Команда Найти из Главного меню позволяет найти: ...

    Все файлы с указанным содержимым.
    Все файлы, созданные за последнюю неделю.
    Все ответы правильные.
    Все файлы указанного типа.

  6. Чтобы в приложении Мой компьютер скопировать файл из папки А в папку В, необходимо:

    Выделить файл в папке А, открыть папку В и выбрать команду Правка - Копировать.
    Выделить файл в папке А, выбрать команду Правка - Копировать, открыть папку В и выбрать команду Правка - Вставить.
    Выделить файл в папке А, открыть папку В и выбрать команду Правка - Вставить.
    Выделить файл в папке А, выбрать команду Правка - Вставить, открыть папку В и выбрать команду Правка - Копировать.

  7. Чтобы найти нужный файл или папку с помощью главного меню, нужно: ..

    Нажать кнопку Пуск и выбрать команду Выполнить.
    В главном меню Windows нет подходящей команды.
    Нажать кнопку Пуск и выбрать команду Документы.
    Нажать кнопку Пуск и выбрать команду Найти

  8. Сочетание клавиш, с помощью которого можно удалить папку или файл, минуя корзину. ...

    SHIFT+DEL.
    ALT+CTRL
    CTRL+DEL.
    ALT+CTRL+DEL
    ALT+DEL

  9. Если программа перестала реагировать на Ваши действия, необходимо: ...

    Нажать клавишу ESC.
    Нажать клавиши ALT+F4
    Нажать клавиши ALT+CTRL+DEL.

  10. После нажатия кнопки Вырезать на панели инструментов приложения ...

    Удаляется все содержимое текущей папки.
    Удаляется выбранный объект (файл или папка).
    Выбранный объект удаляется в буфер обмена.
    Выбранный объект перемещается в Корзину.
