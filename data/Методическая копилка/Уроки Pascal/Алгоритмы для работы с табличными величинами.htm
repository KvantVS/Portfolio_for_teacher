ТЕМА 1.16. Алгоритмы для работы
с табличными величинами

Массивы

 

          Многие задачи, которые решаются с помощью компьютера, связаны с обработкой больших объемов информации. Для удобства обработки информация часто сводится в таблицу.  Таблицы бывают линейные и прямоугольные.

          Каждому значению, или элементу таблицы, соответствует его порядковый номер, и наоборот, стоит задать порядковый номер, и сразу ясно, о каком элементе таблицы идет речь.

Линейная таблица

                                                                 Таблица 1

1

2

3

1,5

-6

Данная таблица имеет 5 элементов. Пятый элемент таблицы имеет значение 8, а третий – значение –6. Значение 1,5 принадлежит второму элементу, значение 12 – четвертому элементу таблицы.

Очевидно, что при хранении таблицы порядковые номера хранить нет необходимости: зная начало нумерации, можно путем отсчета найти любой элемент. Кроме того, полезно знать и самый большой порядковый номер, т.к. это позволяет определить заранее размер таблицы.

Таким образом, если некоторая величина является линейной таблицей, нужно задать тип элементов таблицы, ее имя, начальный и конечный порядковые номера таблицы.

В алгоритмах табличные величины описываются следующим образом: служебное слово, указывающее тип (цел, вещ, лит), затем служебное слово таб (таблица), имя таблицы, за которым стоят в квадратных скобках начальный и конечный порядковые номера ее элементов, разделенных двоеточием. Например:

вещ таб А [1:5]

цел  таб D [2:20]

лит  таб К [20:30]

 

Работа с табличной величиной сводится к работе с ее элементами. Для того, чтобы указать, какой элемент в данный момент используется, достаточно указать его порядковый номер – индекс.

Индекс заключается в квадратные скобки и помещается вслед за именем таблицы.

Для таблицы 1 (пусть ее имя А):

                   А[5] = 8

                   A[2] = 1,5

Пятый элемент имеет значение 8, второй – 1,5.

Для работы на компьютере с табличными величинами используются массивы.

Массив – это упорядоченная по номерам совокупность значений, объединенных общим именем  и типом.

При формировании массива компьютер отводит последовательность расположенных друг за другом ячеек, имеющих одинаковые имена.

Элементы массива нумеруются. Номер элемента (индекс) – характеризует положение элемента относительно начала массива.

Массивы в Pascal описываются в разделе описания переменных  с помощью служебного слова array:

 

var имя массива:array[нач.индекс .. макс.индекс] of  тип элемента

 

Например, var a: array[1 .. 4] of  byte.

 

Начальным индексом элементов данного массива считается 1. При выполнении оператора array[1..4] of byte Pascal выделит 4 ячейки под массив.

 

а(1)

a(2)

a(3)

a(4)

Пример 65. Определить для таблицы А сумму положительных, количество нулевых, произведение отрицательных элементов.

 

Дано:          таблица А[1:N], размерность таблицы N

Требуется: S – сумма положительных, Р – произведение отрицательных,

К – количество нулевых элементов таблицы А

Связь:         Для обработки таблиц любой размерности необходимо задать в качестве исходного данного значение N, определяющее количество элементов таблицы. В качестве примера используем таблицу А[1:6].

Для определения результатов следует просматривать поочередно все элементы таблицы и делать выбор:

а) если элемент А[i] положителен (A[i]>0), то считать его слагаемым для суммы S=S+A[i];

б) если элемент А[i] отрицателен (A[i]<0), то считать его сомножителем для произведения Р=Р × A[i];

в) если A[i]=0, то значение К=К+1.

алг S, K, P (арг цел N, вещ таб А[1:N], рез вещ

                    S, P, цел K)

 

 

нач цел I

       ввод N

        для I от 1 до N

           нц

               выбор

                         при A[I] > 0 : S := S + A[I]

                         при A[I] = 0 : K := K + 1

                         при A[I] < 0 : P := P × A[I]

               все

            кц

       вывод S, K, P

 

кон

if a[i]=0 then k:=k+1;

if a[i]<0 then p:=p*a[i];

СОРТИРОВКА МАССИВОВ

 

          Процедура перестановки элементов массива по возрастанию или убыванию называется сортировкой массива.

          Для сортировки массивов создано множество алгоритмов. Рассмотрим два из них.

Сортировка выбором.

1. Определим максимальный элемент в таблице и запомним его индекс, например, как переменную С.

2. Обменяем значение последнего элемента А[N] и элемента А[С]. Тем самым, в последней клетке таблицы окажется самый большой элемент.

3. Найдем максимальный элемент среди A[1], …, A[N-1] и обменяем его с  А[N-1] и т.д.

нач    цел I, J, C, вещ MAX, R

   нц

для I от 1 до J

   нц

то MAX:=A[I]; C:=I

все

   кц
